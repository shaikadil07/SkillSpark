# 🚀 SkillSpark – Ignite Your Daily Learning Streak

SkillSpark is an interactive web-based learning platform designed to help users build consistent learning habits through daily Python and Java programming topics. The platform combines micro-learning with gamification features such as streak tracking, motivational messages, and achievement animations to make learning engaging and rewarding.

---

## 📖 About the Project

SkillSpark provides users with daily programming topics in Python and Java through an easy-to-use interface. Users can explore concepts one topic at a time, maintain daily learning streaks, and stay motivated through visual rewards and progress tracking.

The main goal of the project is to encourage continuous learning and help users strengthen their programming fundamentals through small but consistent daily efforts.

## ⚙️ How It Works

1. User logs into the application.
2. Daily Python and Java topics are displayed.
3. User navigates through learning cards.
4. Progress is monitored throughout the session.
5. Daily streaks are awarded after completing learning goals.
6. Motivational messages and animations encourage continuous learning.
